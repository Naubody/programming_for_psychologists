# Programming for Psychologists 2024  

## Course Details  
**Course Coordinator:** Matthias Nau  
**Teaching Assistant:** Anna van Harmelen  
**Contact:** [m.nau@vu.nl](mailto:m.nau@vu.nl)  

This repository contains the course materials for the course **"Programming for Psychologists"** at the Vrije Universiteit Amsterdam (P_MPROPSY). This 6 EC course will teach you a new superpower: **Programming in Python!**  

Over the course of 2 months, you will learn to think like a programmer and use Python for **experimental design, data analysis, and visualization**. You will be introduced to essential concepts, data structures, and libraries relevant to cognitive neuroscience research. Additionally, you will gain experience in Open Science principles, **code review**, and ensuring research is **reproducible, accessible, and transparent**.

## Course Structure  
The course is organized into **7 modules**, each consisting of a **lecture** and **two practicals**. All sessions will be **in-person**, and lectures may be recorded.  

### Lectures  
Lectures focus on **conceptual understanding**, discussing the importance of programming for psychologists, Python fundamentals, coding best practices, and Open Science principles.  

**Location:** Room MF-A611  
**Schedule:** Every other Monday, 3:30-5:15 PM  

### Practicals  
Practicals focus on **hands-on coding**, covering basic Python operations, experimental design, data analysis, and GitHub for version control.  

**Locations & Schedule:**  
- **Monday:** Room HG-11A22, 3:00 PM  
- **Wednesday:** Room OZW-4A05, 10:00 AM  
- Each session lasts **2-2.5 hours**

## Home Assignment  
The main assignment involves **writing code** for **data analysis and visualization of MRI data**.  

- You will write code and include a brief statement explaining the best practices you followed.  
- The code will be uploaded to **GitHub** for **peer review**, where students will provide feedback on each other’s work.  
- More details will be provided in **November**.  

**The home assignment counts for 65% of your final grade.**  

---

## Exam: Code Review  
At the end of the course, you will take an **exam** that includes:  
- Analyzing code snippets and answering related questions  
- Conceptual questions on lecture and practical content  
- A mix of **open-ended** and **multiple-choice** questions  

**The exam counts for 35% of your final grade.**

## How to Become an Expert P(s)ychologist  
This course provides a strong foundation, but mastering programming takes **practice and time**. Below are some **recommended resources** to help you grow beyond the course:  

- **[Automate the Boring Stuff](https://automatetheboringstuff.com/)** – A beginner-friendly Python course  
- **[The Good Research Handbook](https://goodresearch.dev/)** – Best practices for research coding  
- **[Harvard CS50 Python Course](https://cs50.harvard.edu/python/)** – A more in-depth programming course  
- **[Python for Neuroscientists (eLife)](https://elifesciences.org/articles/53222)** – A helpful guide with practice notebooks  

All these resources are **free and available online**.
